Note: CDN versus "npx create-react-app"?

What should you choose between simple CDN links or "npx create-react-app"? I would certainly recommend the second option for most projects, but for the sake of convenience, I decided to use CDN for this aide-mémoire. At a bare minimum, you should load libraries locally instead of remotely (CDN). This should really improve loading time.

CDN

If you use CDN, you have to load the following libraries.

The last library is useful if you want to force the type of your props.

  • <script src="https://unpkg.com/react@16/umd/react.production.min.js" crossorigin></script>
  • <script src="https://unpkg.com/react-dom@16/umd/react-dom.production.min.js" crossorigin></script>
  • <script src="https://unpkg.com/babel-standalone@6.15.0/babel.min.js" crossorigin></script>
  • <script src="https://unpkg.com/prop-types@15.6/prop-types.min.js" crossorigin></script>
